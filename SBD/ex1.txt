1. Defina e correlacione os conceitos de banco de dados, sistema de banco de dados e de sistema gerenciador de banco de dados.
	R: O conceito de banco de dados, é basicamente um aglomerado de informações reunidas e diretamente relacionadas, que sao armazenadas sobre um dominio específico, ja 
o SBD é um sitema de regitros envolvendo quatro componentes principais, hardware, software e usuario, um SBD possui como objetivo isolar o usuario de datalhes 
mais internos do BD(abstração de dados) e prover independencia de dados as aplicações, com o SGBD é um pouco diferente, pois é um software com recursos específicos para 
facilitar a manipulação de informações de um BD e o desenvolivmento de programas aplicativos como Oracle, Parados, MySQL, Interbase, Sybase, etc..

2. Como os SGBD garantem independência e consistência de dados?
	R: Através dos mecanismos de roll backs por exemplo, para evitar que um estado de inconsistencia seja realizado o roll back devolve a base de dados ao seu estado de 
consistência. E a independência, e garantido porque cada aplicação vê apenas os dados que lhe interessam, Ela nao precisa saber como os dados estão fisicamente armazenados, 
e tambem nao precisa ser modificada caso a estrutura de dados seja alterada.

3. Discuta as vantagens de um SBD quando comparado com um sistema de arquivos.
    R: Quando possuir um conjunto muito grande de dados, é inviavel usar um sistema local de arquivos, pois um SBD possui diversas vantagens como independencia de arquivos, 
ou seja um SBD nao precisa conhecer as estruturas dos dados, nao precisa saber detalhes de como seus dados estao fisicamente armazenados e também possui Redundancia 
controlada,os dados armazenados possuem consistencia e segurança entre muitas outras vantagens que superam os arquivos locais convencionais, porem um sistema de arquivos para 
dados pequenos costuma ser mais vantajoso pela questão de custo e facilidade de gerenciamente.
